StudioCMS Integration config options schema Reference

```ts twoslash title="studiocms.config.mjs"
import { defineStudioCMSConfig } from 'studiocms/config';
// ---cut---
export default defineStudioCMSConfig({
  imageService: {
    cdnPlugin: 'cloudinary-js'
  },
});
```

## `cdnPlugin`

`cdnPlugin` is a enum that is used to determine which CDN plugin to use for images if enabled.

- **Type:** `'cloudinary-js'` | `undefined`
- **Default:** `undefined`
